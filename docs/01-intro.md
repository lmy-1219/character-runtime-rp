# 开篇 · 这套系统是什么

起因很简单：打完《赛博朋克 2077》，所有结局我都不满意。于是从零开始闭门造车，一路踩坑硬解构，搞到现在——Johnny 总算能在我搭的这座文本夜之城里像个活人一样过日子了，文风也基本调到了我想要的样子。

动手之前我没看过任何经验贴，纯闭着眼睛干。要是已经有大佬总结过更省事的解法，欢迎指路、拍砖、交流。技术和逻辑不到位的地方，多担待。

它不教 AI「模仿一个叫 Johnny Silverhand 的人」，而是给 AI 一套角色的底层运行逻辑，让它每一轮自己推演出——这个角色此刻该说什么、做什么、不做什么。

打个比方。以前做 RP，像给 AI 演员递剧本：「这段你演愤怒。」现在这套，像是给演员一本人物小传 + 心理分析 + 行为准则，让他自己想明白「我为什么愤怒、我愤怒时一般怎么做、闹完之后又怎么收场」，然后放他自由发挥。

实测 600+ 轮下来（角色卡中途重构了不下三次），体感是这样：

- **立体**——他不像在「表演情绪」，像在「过自己的日子」。
- **有张力**——言行有前因、有后果、有矛盾、有惯性，不是随机触发。
- **稳**——不会写着写着变成温柔老父亲，也不会写着写着变成纯喷子。
- **能长跑**——连续两百轮没有明显记忆缺失，性格不走形，估计撑百万字级没问题。

实话实说，缺陷一堆，本质是**成本和质量的取舍**：单条贵（每次生成要先「算」一轮角色状态，吃 token）；吃算力（小模型跑不动，得上强推理大模型、最好开思考模式，基本得用闭源 API）；复杂（学习曲线陡，跑通后维护半自动，但前期搭建费功夫）；慢（保质量的代价）。

**适合谁：** 热爱某个角色（尤其是 IP 角色）、特别想让他在自己的世界里活着、对角色精细度和模型表现都有要求的 RP 玩家。

### 读之前，几句话摆前面

1. **这套逻辑不是万能模板。** 我所有经验都来自这一张卡：高复杂度、脾气差、高创伤低敏感、阴阳怪气拉满、日常幽默嘴炮、高压时果决——一个安那其革命家，前摇滚巨星。这种配置直接决定了我写的规则、纠偏方向、纠偏强度、连世界书怎么分层都带着他的烙印。你要演温柔治愈、正太、纯反派、或完全不沾幽默的角色，**思路可以参考，直接套大概率出问题。**

2. **我在本文里举的例子是我的论据，不是提示词模板。** 所有示例都基于我的原提示词和我这个角色类型；底层逻辑大概率能通用，但把例子里的原句直接搬进你的卡，效果很可能不理想——哪些是逻辑、哪些是我这张卡的私货，得自己分辨。

3. **有些部分我没做多轮实测。** 凡是只跑过轻量验证、没经过长程多轮压测的设计，我都会在那一节明确标注——那部分是思路，不是结论。
