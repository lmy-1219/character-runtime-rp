# inspiration · 灵感库

这里是一个尚未进入架构的跨领域理论线索池。里面的东西都还没被验证、也还没落进 `docs/` 或 `lab/` 的任何设计——只是我在别的领域看到、觉得可能和这套 RP 架构有接口的线索，先记下来，留着以后慢慢查、慢慢想。按「研究现场」看，不是结论。

分两类持续记录，方便后面各自往下追加：

- [理论线索](theory-leads.md) —— 跨领域的理论方向，我怀疑它们能给「什么算发生过、它如何存活到下一轮」「受控泄漏 / 嵌套扮演」这类问题提供借鉴，但还没深挖。
- [模型观察](model-notes.md) —— 各个模型在我这套架构里的实际表现记录（保姆倾向、张力消解、术语触发之类），零散攒着，供选型和纠偏参考。

以后新增灵感，按性质归到这两个文件里；开新类别时再在这份索引里加入口。
